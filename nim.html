My partner and I used Python to create a game. We chose Nim to be our game. I coded the random message genarator idepedently. Surprisingly, this was a difficult task. I had to find a way to make a random number genarator with variables, and then find a way to display the messages themselves and not the variable. To code it, I assigned multiple variables to multiple messages, and had them be chosen at random. I gave it the name "messages" and had each turn display a randomized message. By giving it the name "messages" and setting "messages" equal to the answer, I was able to display the messages, overcoming my problem.
